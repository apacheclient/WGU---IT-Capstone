## 📋 Project Summary

Texas Grocers operates ~300 stores across Texas with 85,000 associates. Their legacy desktop-only system created major productivity barriers.

**I designed, developed, and implemented** a modern **centralized workforce portal** — a secure, responsive web application serving as a **Single Sign-On (SSO)** gateway with robust **Role-Based Access Control (RBAC)**.

**Key Outcomes:**
- Productivity increased **>80%** in pilot stores
- Significant reduction in helpdesk tickets
- 95%+ uptime
- Zero critical security vulnerabilities

---

## ✨ Core Features

- Responsive, branded UI/UX (mobile-first)
- Single Sign-On (SSO)
- Granular Role-Based Access Control (RBAC) with auditing
- API integrations with legacy systems
- Cross-device support on corporate WLAN
- Enterprise security (MFA, encryption, etc.)
- Personalized self-service dashboard

---

## 🛠️ Technology Stack

- **Frontend:** React.js
- **Backend:** Node.js
- **Auth & Integration:** SSO, REST APIs, custom middleware
- **Security:** MFA, RBAC, audit logging
- **Methodology:** Hybrid Waterfall + Agile
