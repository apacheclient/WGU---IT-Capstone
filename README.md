## 📋 Project Summary

Texas Grocers operates approximately 300 stores across Texas with 85,000 associates. Their legacy desktop-centric system created major productivity barriers — limited WLAN access, poor cross-device support, multiple logins, and administrative overload.

**I designed, developed, and successfully implemented** a modern **centralized workforce portal** — a secure, responsive web application that functions as a **Single Sign-On (SSO)** gateway with robust Role-Based Access Control (RBAC).

**Key Outcomes:**
- Productivity increased **>80%** in pilot stores
- Significant reduction in helpdesk tickets
- 95%+ uptime achieved
- Zero critical security vulnerabilities
- High user satisfaction and adoption

---

## ✨ Core Features

- Responsive, branded UI/UX optimized for mobile, tablet, and desktop
- Single Sign-On (SSO) authentication
- Granular Role-Based Access Control (RBAC) with audit logging
- Seamless API integrations with legacy inventory, scheduling, training, and HR systems
- Full cross-device compatibility on corporate WLAN
- Enterprise security (MFA, encrypted sessions, strict network controls)
- Personalized self-service dashboard
- Analytics and performance monitoring

---

## 🛠️ Technology Stack

- **Frontend:** React.js
- **Backend:** Node.js
- **Authentication & Integration:** SSO, RESTful APIs, custom middleware
- **Security:** MFA, encryption, RBAC, comprehensive audit logs
- **Methodology:** Hybrid Waterfall + Agile
